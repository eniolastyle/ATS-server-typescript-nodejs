ATS API Description

The Application Tracking System (ATS) API is a tool for managing job applications, tracking candidate information, and streamlining the hiring process. It provides a set of endpoints to interact with application data, enabling you to create, retrieve, update, and delete applications seamlessly.

API Endpoints

Endpoint Description
/ Root endpoint with a welcome message and API description.
/api-docs Swagger documentation for the API.
/applications Get a list of all job applications.
/applications (POST) Create a new job application.
/applications/{id} Get a specific job application by ID.
/applications/{id} (PUT) Update an existing job application by ID.
/applications/{id} (DELETE) Delete a job application by ID.

For detailed documentation and testing, you can visit the Swagger documentation.
